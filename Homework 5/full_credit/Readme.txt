To compile and run use command make

To use the load function teh format for input in the save file must be in the form
dd|mm|yyyy|Transaction amount|Employee Name

Note : the date has to be entered in the form mm/dd/yyyy and the time hh:mm:ss
The month value must have a maximum of 12 and minimum of 1
The date value must have a maximum of 31 and a minimum of 1
The year value must have a maximum of 12 and a minimum of 1
The hour miniute and second values must have a maximum of 60 and minimum of 1

Note if input is invalid for menu the program will terminate
